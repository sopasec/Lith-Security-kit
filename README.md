LITH (Lightweight Information & Threat Helper)Is a windows multi tool kit for anything need
I hope with LITH you can fix any problem you have easily.

Features

System Tools
- CPU usage monitor
- RAM usage monitor
- Disk health checks
- Running process viewer
- System information

Network Tools
- Active connection viewer
- Open port detection
- LAN device scanner
- Network traffic monitor
- Ping connectivity test

Security Tools
- Suspicious file scanner
- Startup program viewer
- Windows system file checker
- Windows Defender quick scan

Monitoring
- Live CPU monitor
- Live RAM monitor
- Network activity graphs

Reporting
- Security logs
- Custom log messages
- Full system security report generator

Requirements 

windows 10 or plus

## How to Run

1. Download lith4.bat
2. right click
3. Run as admin
